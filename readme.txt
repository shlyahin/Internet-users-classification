Учебный проект в рамках специализации "Машинное обучение и анализ данных" (Coursera - МФТИ, Яндекс).

В проекте осуществлялась идентификация пользователя по последовательности из нескольких веб-сайтов, посещенных подряд на основе данных с прокси-серверов Университета Блеза Паскаля.

Educational project within the framework of the specialization "Machine learning and data analysis" (Coursera - MIPT, Yandex).



The project carried out user identification by a sequence of several website visits, based on data from Blaise Pascal University proxy servers.
